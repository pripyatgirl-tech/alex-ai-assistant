# Alex — Local AI Assistant

Локальный AI-ассистент на базе Ollama с кастомной личностью.

## Что умеет
- Поддерживает историю диалога между сессиями
- Загружает кастомную личность из файла
- Система памяти - запоминает информацию о пользователе
- Защита от повторяющихся ответов
- Очистка артефактов токенизации

## Стек
- Python 3
- Ollama
- Qwen (кастомная модель через Modelfile)
